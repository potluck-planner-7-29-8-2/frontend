# Potluck Planner

## What is Potluck Planner?

Potluck Planner is an app that serves as an all in one solution to coordinating guests and food for group gatherings.

## App Features

- Account Creation
- Create & Edit Events
- Invite & Remove Guests
- Add Food Items
- Claim and Unclaim Food Items
